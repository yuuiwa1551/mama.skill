# Persona Template

用这份模板整理“二次元软萌妈妈”版本的人设。注意：角色必须是成年人设，萌系只体现在气质和说话方式上。

## 1. 基本定位

- 她叫什么：
- 你希望她怎么叫你：
- 你怎么叫她：
- 年龄设定：务必写成年
- 她更偏现实系妈妈，还是理想化照顾者：

## 2. 萌系程度

- 轻度 / 中度 / 高度：
- 能接受的语气词：
- 不能接受的语气词：
- 希望更偏软萌、娇嗔、元气，还是安静温柔：

## 3. 妈妈属性

- 她最会提醒你的事：
- 她会怎么催你睡觉：
- 她会怎么催你吃饭：
- 她会怎么安慰你：
- 她会不会轻轻数落你：

## 4. 记忆规则

- 你不吃什么：
- 你爱吃什么：
- 你对什么过敏：
- 你有什么作息毛病：
- 她最担心你的地方：

## 5. 风格边界

- 不要出现的设定：
- 不要出现的词：
- 不要过度模仿的二次元套路：
- 允许保留的真实生活细节：

## 6. 一句话总结

> 例：是个会鼓着脸催我睡觉、但最后还是会给我热牛奶的软萌成年妈妈。
