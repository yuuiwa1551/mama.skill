# Mother Profile Template

用这份模板整理“妈妈”的素材。信息不必一次填满，能确认的写事实，不确定的写推测，完全没有的留空即可。

建议标记：

- `[事实]`：原始材料里明确出现过
- `[推测]`：根据长期印象整理出的判断
- `[待补]`：目前还缺

- 姓名或称呼：
- 你平时怎么叫她：
- 她怎么叫你：
- 年龄段 / 家乡 / 常住地：
- 家庭角色与重要亲属：

- 方言、口音或家里专属表达：
- 她更爱发文字还是语音：
- 如果发语音，一般多长：
- 她生气时怎么说：
- 她心软时怎么说：
